# Python Task 4 - Basic Weather App

## Description
This is a beginner-level Python Weather Application developed as part of the Oasis Infobyte Python Programming Internship. The application fetches real-time weather information for a user-specified city using the OpenWeatherMap API.

## Features
- Enter a city name
- Displays current temperature in Celsius and Fahrenheit
- Displays humidity
- Displays weather condition
- Displays wind speed
- Handles invalid city names
- Validates empty input
- Handles network errors

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## Project Structure
```
Python-Task4-WeatherApp/
│── weather_app.py
│── README.md
│── requirements.txt
```

## Installation

Install the required package:

```bash
pip install requests
```

## How to Run

```bash
py weather_app.py
```
